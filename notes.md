class methods
- uses 'static' keyword
- cannot use 'this' keyword
- available to any instance of class itself
- can be made available to other classes
- does NOT require object of class for methods to be called

- instance methods - methods that affect a particular object should be defined
  as instance methods
- class methods - provide general capability but does not directly affect an
  object of the class should be declared as class methods


## access modifiers
- for classes you can use:
    - public = accessible by any other class
- for attributes, methods and constructors
    - private = code is only accessible within the declared class
    - protected = code is accessible in same package and subclasses
